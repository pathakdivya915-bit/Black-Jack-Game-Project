# 🃏 Blackjack Game (Python)

A simple yet interactive command-line Blackjack game built using Python. This project demonstrates core programming concepts like loops, conditionals, functions, and basic game logic.

---

## 🚀 Features

* Play Blackjack against the computer (dealer)
* Random card generation
* Realistic Blackjack rules (Ace handling, dealer logic, bust conditions)
* Clean and beginner-friendly code structure
* Replay option to play multiple rounds

---

## 🛠️ Tech Stack

* **Libraries Used:**

  * `random`

## 🎮 How to Play

1. Run the Python file:

   ```bash
   python blackjack.py
   ```

2. You and the dealer will each receive cards.

3. Choose to:

   * **Hit** → Take another card
   * **Stand** → End your turn

4. The dealer plays automatically.

5. The closest score to 21 wins without going over.

---

## 🧠 Game Rules

* Number cards = face value
* Face cards (J, Q, K) = 10
* Ace = 1 or 11 (automatically adjusted)
* If total > 21 → Bust
* Dealer must hit until reaching at least 17
